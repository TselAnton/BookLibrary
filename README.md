# BookLibrary

### Нужно манипулировать сущностью Book со следующими полями:
- Название книги
- Имя автора
- Издание
- Количество страниц
- Номер полки
- Прочитана ли уже книга
- Название цикла
- Закончен ли весь цикл
- Номер книги в цикле

Программа для домашнего пользования

Java 17, to build: mvn clean package
